# 🌿 Mana Coffee | Ritual & Concreto

[![React](https://img.shields.io/badge/React-20232A?style=for-the-badge&logo=react&logoColor=61DAFB)](https://reactjs.org/)
[![Vite](https://img.shields.io/badge/Vite-646CFF?style=for-the-badge&logo=vite&logoColor=white)](https://vitejs.dev/)
[![TypeScript](https://img.shields.io/badge/TypeScript-007ACC?style=for-the-badge&logo=typescript&logoColor=white)](https://www.typescriptlang.org/)
[![Tailwind CSS](https://img.shields.io/badge/Tailwind_CSS-38B2AC?style=for-the-badge&logo=tailwind-css&logoColor=white)](https://tailwindcss.com/)

Onde a tradição milenar coreana do Matcha encontra o pulso brutalista de São Paulo. **Mana Coffee** é uma experiência tátil, visual e sensorial localizada no coração do Bom Retiro e Pinheiros.

## ✨ Características

- 🍵 **Matcha de Origem**: Grãos colhidos no solo vulcânico da Ilha de Jeju.
- 🏗️ **Estética Botanical Brutalist**: Um encontro harmônico entre o concreto honesto e o verde vibrante.
- 📱 **Mobile-First**: Design totalmente responsivo refinado para dispositivos móveis (320px+).
- 🎨 **Fidelidade Visual**: Implementação pixel-perfeita do design system Stitch.
- ⚡ **Performance**: Single Page Application (SPA) ultra-rápida construída com Vite e React.

## 🛠️ Stack Tecnológica

- **Framework**: [React 18+](https://react.dev/)
- **Linguagem**: [TypeScript](https://www.typescriptlang.org/)
- **Build Tool**: [Vite](https://vitejs.dev/)
- **Estilização**: [Tailwind CSS](https://tailwindcss.com/)
- **Roteamento**: [React Router](https://reactrouter.com/)
- **Tipografia**:
  - `Epilogue`: Headlines impactantes (Bold/Black).
  - `Plus Jakarta Sans`: Corpo e labels funcionais.
  - `Noto Serif`: Toques de elegância clássica.

## 🚀 Como Iniciar

1. **Clone o repositório:**
   ```bash
   git clone https://github.com/ffmpenna/mana-coffee.git
   cd mana-coffee/mana-coffee-react
   ```

2. **Instale as dependências:**
   ```bash
   npm install
   ```

3. **Inicie o servidor de desenvolvimento:**
   ```bash
   npm run dev
   ```

4. **Acesse no navegador:**
   `http://localhost:5173`

## 📐 Design System

Este projeto utiliza o conceito **Botanical Brutalist**, caracterizado por:
- **Cores Primárias**: `#34562e` (Mana Green) e `#c4eeb8` (Matcha Soft).
- **Cores de Acabamento**: `#f1f3ed` (Bone White/Sage) e `#efdec0` (Natural Sand).
- **Tipografia**: Headers em caixa alta, tracking tigh e peso Black.
- **Micro-animações**: Transições suaves de hover e rotações sutis em cards.
